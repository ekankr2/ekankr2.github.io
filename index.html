  • 콜드 스타트 시 커넥션 생성 지연으로 타임아웃 발생

  • 또는 MongoDB 같은 NoSQL로 변경하여 커넥션 제한 문제 우회
해결 과정.
  • 1 : NoSQL 전환 vs 고정 서버 전환 비교 후 고정 서버 선택 (데이터 정합성, 비용 고려)
  • 2 : Lambda/Vercel → AWS ECS Fargate로 마이그레이션
  • 3 : ALB 오토스케일링 구성으로 서버리스와 비슷한 가용성 확보
결과.
  • DB 커넥션 에러 100% 해결
  • 응답 시간, 콜드스타트 문제 해결
  • 서버리스 대비 예측 가능한 성능 확보
